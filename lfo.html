How to use
  1. Wire up: Grab the ⊕ drag to assign handle from an LFO and drag it onto any slider below. Release to connect.
  2. Adjust depth: Drag the depth % label on the floating badge left/right to change modulation intensity.
  3. Chain LFOs: Drag LFO 2 or LFO 3's handle onto LFO 1's Rate or Depth slider. The chain LFO will modulate the target LFO's parameter.
  4. Remove: Click the × on any mod badge to disconnect.
  5. Shape: Click a shape button (SIN / TRI / SAW / SQR…) on any LFO to change its waveform.
  6. Programmatic: widget.connect(element, { depth: 0.7 }) or use the engine API directly.
